# Стресс-менеджмент

> Глоссарий терминов и сокращений: [Открыть](../glossary.md)


Стресс-менеджмент аналитика это система профилактики перегрузки и восстановления, чтобы сохранять качество решений в интенсивной проектной среде.

## Уровни сложности

### Базовый уровень

- распознавать ранние признаки стресса;
- использовать короткие техники саморегуляции;
- ограничивать перегрузку через WIP и паузы.

### Средний уровень

- строить личный протокол восстановления;
- управлять источниками стресса через планирование и коммуникацию;
- применять техники когнитивной переоценки.

### Продвинутый уровень

- системно предотвращать выгорание в команде;
- внедрять рабочие ритуалы устойчивости;
- работать с хроническим стрессом через долгосрочные стратегии.

## Признаки стресса

| Уровень | Признаки |
| --- | --- |
| Когнитивный | снижение концентрации, забывчивость |
| Эмоциональный | раздражительность, тревожность |
| Поведенческий | прокрастинация, резкие реакции |
| Физический | усталость, нарушение сна |

## Краткосрочные техники (5-15 минут)

- дыхание 4-6 или box-breathing 4-4-4-4;
- микро-паузы между контекстами;
- короткая прогулка/растяжка;
- mindfulness check-in: "что сейчас под моим контролем".

## Долгосрочные стратегии

- time-blocking и ограничение WIP;
- регулярная физическая активность;
- планирование восстановления в календаре;
- делегирование и управление ожиданиями;
- периодические ретроспективы нагрузки.

## Профилактика выгорания

| Риск | Профилактика |
| --- | --- |
| хронические переработки | жесткий лимит рабочего дня + буфер в планах |
| постоянный контекст-switch | deep work окна и правила уведомлений |
| конфликт ролей | явный RACI и эскалация при перегрузе |
| отсутствие контроля | weekly planning и daily review |

## План личной устойчивости (resilience)

1. Определить 3 ключевых стресс-триггера.
2. Для каждого подготовить реакцию 1-го уровня (быстрый ответ).
3. Зафиксировать долгосрочную профилактику.
4. Проверять план раз в неделю.
