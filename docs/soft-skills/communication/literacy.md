# Грамотность

Грамотность аналитика это точность письменной и устной речи, единая терминология и корректный тон коммуникации.

## Уровни сложности

### Базовый уровень

- писать грамотно и структурно;
- избегать двусмысленных формулировок;
- использовать единый словарь терминов.

### Средний уровень

- адаптировать язык под аудиторию (бизнес/тех);
- формулировать требования и решения однозначно;
- поддерживать стандарт качества документов.

### Продвинутый уровень

- выстраивать редакционную политику команды;
- развивать культуру peer review текста;
- снижать риски miscommunication на уровне организации.

## Правила практической грамотности

- одно предложение = одна мысль;
- избегать расплывчатых слов: "быстро", "удобно", "при необходимости";
- отделять факты, решения и действия;
- использовать терминологию из согласованного глоссария.

## Мини-чеклист документа

- цель документа понятна из первых 3-5 строк;
- термины определены и не противоречат другим разделам;
- есть явные решения/owner/срок;
- текст корректен по орфографии и пунктуации.

## Пример

Нечетко:

"Система должна быстро обрабатывать заявки".

Четко:

"95% заявок должны переходить в статус `processed` не позднее 2 минут в рабочее время".
