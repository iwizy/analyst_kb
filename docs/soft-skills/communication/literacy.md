# Грамотность

Грамотность - это точность и ясность письменной речи. Для аналитика это
напрямую влияет на качество требований, договоренностей и документации.

## Что включает навык

- корректность языка и терминов;
- логичная структура текста;
- однозначность формулировок;
- соответствие текста целевой аудитории.

## Практические правила

- используйте простые предложения и активный залог;
- избегайте оценочных формулировок без критериев;
- выносите решения и действия отдельными пунктами;
- проверяйте документ на двусмысленность перед публикацией.

## Лучшие практики

- применять единый глоссарий терминов;
- использовать шаблоны для требований и протоколов;
- выполнять peer-review критичных документов;
- регулярно редактировать устаревшие материалы.

## Типичные ошибки

- расплывчатые слова: "быстро", "удобно", "при необходимости";
- смешение требований, обсуждений и решений в одном абзаце;
- несогласованные термины между документами.

## Чек-лист качества

- Текст читается без дополнительных устных пояснений.
- Требования проверяемы и однозначны.
- Термины согласованы с глоссарием проекта.

## Упражнение на неделю

1. Выберите один рабочий сценарий, где навык критичен (переговоры, конфликт, review, планирование).
1. Примените технику из статьи осознанно минимум в двух коммуникациях.
1. Зафиксируйте результат по схеме: `ситуация -> действие -> эффект -> что улучшить`.
