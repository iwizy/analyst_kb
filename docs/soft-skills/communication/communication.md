# Умение общаться

Умение общаться для аналитика - это способность передавать сложную информацию
понятно, адресно и без потери смысла.

## Базовые принципы

- ясность: одна мысль - один тезис;
- контекст: сначала цель, затем детали;
- адресность: учитывайте роль и ожидания собеседника;
- обратная связь: проверяйте, что вас поняли правильно.

## Практический сценарий

На встрече по интеграции аналитик:

1. формулирует проблему в 1-2 предложениях;
1. показывает варианты решения с последствиями;
1. фиксирует договоренности и открытые вопросы.

## Лучшие практики

- используйте структуру `контекст -> вариант -> решение -> next step`;
- отделяйте факты от интерпретаций;
- фиксируйте результат коммуникации письменно;
- перефразируйте услышанное при высокой неопределенности.

## Типичные ошибки

- длинные объяснения без фокуса на решении;
- неуточненные термины между командами;
- отсутствие явного завершения разговора (кто делает что и когда).

## Чек-лист качества

- Собеседник понимает, какое решение требуется.
- Договоренности зафиксированы и проверяемы.
- Нет разночтений в ключевых терминах.
