# Тайм-менеджмент

Тайм-менеджмент аналитика - это управление фокусом, а не только календарем.

## Практические техники

- Eisenhower Matrix (важно/срочно);
- Time Blocking;
- Pomodoro для задач с высоким порогом входа;
- weekly review и replanning.

## Шаблон дня аналитика

| Блок | Время | Тип работы |
| --- | --- | --- |
| Deep work 1 | 09:00-10:30 | требования/моделирование |
| Коммуникации | 10:30-12:00 | синки, уточнения |
| Deep work 2 | 13:00-14:30 | аналитика/документация |
| Операционный контур | 14:30-16:00 | review, ответы, backlog |

## Типичные ошибки

- календарь без буферов;
- реактивный режим весь день;
- отсутствие недельной приоритизации.

## Упражнение

Проведите time audit на 5 рабочих дней и перераспределите минимум 20% времени в deep work.
