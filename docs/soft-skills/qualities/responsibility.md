# Ответственность

Ответственность в аналитике - это способность доводить обязательства до
результата и прозрачно управлять отклонениями.

## Что включает навык

- соблюдение договоренностей по срокам и качеству;
- своевременная коммуникация рисков и задержек;
- контроль полноты и актуальности артефактов;
- принятие ownership за результат, а не только за задачу.

## Практические правила

- фиксировать обязательства письменно;
- не скрывать проблемы до последнего момента;
- заранее предлагать план восстановления при отклонениях;
- закрывать задачу только при выполненных критериях готовности.

## Типичные ошибки

- "молчаливые" срывы сроков;
- перекладывание проблемы на внешние факторы без действий;
- формальное завершение задач без фактического результата.

## Чек-лист качества

- Обязательства прозрачны и проверяемы.
- Отклонения эскалируются вовремя.
- Результат соответствует согласованным критериям.
