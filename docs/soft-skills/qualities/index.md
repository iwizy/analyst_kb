# Личностные качества

Личностные качества определяют, как аналитик действует в неопределенности,
реагирует на изменения и доводит решения до результата.

## Упражнение на неделю

1. Выберите один рабочий сценарий, где навык критичен (переговоры, конфликт, review, планирование).
1. Примените технику из статьи осознанно минимум в двух коммуникациях.
1. Зафиксируйте результат по схеме: `ситуация -> действие -> эффект -> что улучшить`.
