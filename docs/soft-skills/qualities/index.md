# Личностные качества

Личностные качества определяют, как аналитик работает в неопределенности, принимает решения и влияет на результат команды.

## Общая модель развития

### Базовый уровень

- понимать значение качества и его поведенческие индикаторы;
- применять базовые упражнения;
- фиксировать наблюдения в рабочем дневнике.

### Средний уровень

- применять качество в реальных проектных ситуациях;
- получать 360-feedback;
- измерять прогресс через поведенческие KPI.

### Продвинутый уровень

- стабильно демонстрировать качество в сложных контекстах;
- помогать развивать качество у коллег;
- связывать личный рост с результатом команды.

## Шкала самооценки (1-5)

| Уровень | Поведенческий индикатор |
| --- | --- |
| 1 | применяет редко, в основном реактивно |
| 2 | применяет в простых случаях |
| 3 | применяет стабильно в типовых задачах |
| 4 | применяет в сложных ситуациях |
| 5 | системно влияет на команду и процесс |

## Метрики прогресса

- количество инициатив, доведенных до результата;
- доля решений с анализом альтернатив;
- скорость адаптации к изменению приоритетов;
- качество обратной связи от коллег и стейкхолдеров.
