# Критическое мышление

Критическое мышление - навык проверки предположений, аргументов и источников до
принятия решения.

## Что помогает мыслить критически

- проверка источников и качества данных;
- поиск контраргументов к собственной позиции;
- анализ когнитивных искажений;
- разделение мнений и фактов.

## Практический алгоритм

1. Сформулировать гипотезу.
1. Определить, какие данные ее подтверждают/опровергают.
1. Проверить альтернативные объяснения.
1. Зафиксировать решение и уровень уверенности.

## Типичные ошибки

- confirmation bias (поиск только подтверждений);
- опора на авторитет вместо данных;
- принятие решения под давлением срочности без проверки.

## Чек-лист качества

- Для решения есть подтверждающие данные.
- Рассмотрены альтернативные версии.
- Ограничения и степень уверенности зафиксированы.
