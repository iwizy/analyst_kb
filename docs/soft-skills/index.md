# Софт-скилы

Раздел посвящен навыкам, которые определяют качество работы аналитика в
команде: коммуникации, взаимодействие с заказчиком, личная эффективность и
личностная устойчивость.

## Почему это критично для аналитика

- даже хорошие требования не работают без качественной коммуникации;
- софт-скилы снижают количество конфликтов и потерь контекста;
- именно эти навыки отличают исполнение задач от системного влияния на результат.

## Как использовать раздел

1. Определите 1-2 навыка, которые сейчас ограничивают ваш результат.
1. Примените практики из статьи в ближайшем рабочем цикле.
1. Оцените эффект по конкретным индикаторам (сроки, качество коммуникаций,
   количество доработок).
