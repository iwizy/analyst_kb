# Работа в команде

> Глоссарий терминов и сокращений: [Открыть](../glossary.md)


Высокая командная эффективность строится на доверии, психологической безопасности и общей цели. Аналитик влияет на это через прозрачность, координацию и качество взаимодействия.

## Уровни сложности

### Базовый уровень

- понимать роли и зоны ответственности;
- поддерживать прозрачную коммуникацию;
- работать по общим правилам команды.

### Средний уровень

- участвовать в распределении ответственности (RACI);
- поддерживать ритуалы команды (daily, retro, refinement);
- помогать в разрешении межфункциональных конфликтов.

### Продвинутый уровень

- усиливать культуру high-performing team;
- внедрять servant-leadership подход;
- развивать систему peer feedback и continuous improvement.

## High-Performing Team

Ключевые признаки:

- доверие и психологическая безопасность;
- единая цель и прозрачные приоритеты;
- самостоятельность при ясных ограничениях;
- регулярная обратная связь и адаптация.

## Роль аналитика в команде

| Контур | Роль аналитика |
| --- | --- |
| Product | sparring-партнер PO в детализации ценности |
| Engineering | уточнение требований и ограничений |
| QA | проверяемость AC, поддержка тест-дизайна |
| Architecture | согласование системных компромиссов |

## Практики взаимодействия

- активное слушание и открытые вопросы;
- короткие синхронизации по блокерам;
- регулярные ретроспективы с action items;
- peer feedback и 1-on-1;
- парное принятие решений по сложным вопросам.

## Team charter (шаблон)

| Поле | Пример |
| --- | --- |
| Общая цель | сократить lead time требований на 20% |
| Принципы работы | transparency, no blame, ownership |
| Роли и ожидания | RACI по ключевым процессам |
| Ритмы | daily, weekly sync, retro |
| Правила эскалации | что и когда эскалируем |

## Динамика команды

- Forming: знакомство и правила.
- Storming: конфликты и уточнение ролей.
- Norming: рабочие договоренности.
- Performing: устойчивый результат.

## Упражнения

- ежемесячный командный health check;
- 1-on-1 по шаблону "что помогает/мешает";
- командное review RACI по изменениям процесса.
