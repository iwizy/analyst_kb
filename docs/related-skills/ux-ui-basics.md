# UX/UI основы

Базовые UX/UI навыки помогают аналитику проектировать требования так, чтобы решение было не только реализуемым, но и удобным для пользователя.

## Что важно знать

- usability heuristics (Нильсен);
- user flow и сценарии;
- wireframes и прототипы;
- accessibility basics (контраст, навигация, фокус).

## Практический процесс

1. Описать пользовательскую цель и pain points.
1. Построить user flow.
1. Создать low-fidelity прототип.
1. Проверить сценарий на 3-5 пользователях/экспертах.
1. Зафиксировать изменения в требованиях.

## Пример

Сценарий: регистрация пользователя.

- текущая проблема: 30% пользователей отваливаются на шаге подтверждения телефона;
- гипотеза: уменьшить количество полей и добавить пошаговый прогресс;
- результат теста прототипа: drop-off снизился до 18%.
