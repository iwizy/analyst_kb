# BI и визуализация

> Глоссарий терминов и сокращений: [Открыть](../../glossary.md)


BI помогает аналитику донести решение, а не просто показать график. Цель дашборда это управленческое действие.

## Уровни сложности

### Базовый уровень

- выбирать тип графика под вопрос;
- строить 1 экран = 1 цель;
- избегать визуального шума.

### Средний уровень

- собирать KPI-дашборд с drill-down;
- проектировать фильтры по сегментам;
- описывать определения метрик и владельцев.

### Продвинутый уровень

- стандартизировать BI-глоссарий и metric layer;
- строить аналитический контур для product/process review;
- внедрять data governance и аудит использования дашбордов.

## Принципы визуализации

- Data-Ink ratio: минимум декоративных элементов, максимум смысла.
- Сначала контекст (baseline/target), затем отклонение.
- Единые цвета для одинаковых сущностей и статусов.
- Явные подписи осей и единиц измерения.

## Как выбрать тип графика

| Вопрос | График | Ошибка |
| --- | --- | --- |
| Как меняется метрика во времени? | line chart | столбцы на длинном периоде |
| Как сравнить категории? | bar chart | pie chart с 10+ сегментами |
| Где узкое место воронки? | funnel chart | таблица без долей перехода |
| Как связаны две величины? | scatter plot | line chart без временной оси |

## Мини-шаблон KPI-дашборда

| Блок | Содержимое |
| --- | --- |
| Заголовок | цель дашборда и период |
| KPI | значение, baseline, target, delta |
| Диагностика | разрезы по каналам/регионам/продуктам |
| Действие | гипотеза и владелец следующего шага |

## Кейc: снижение drop-off в регистрации

- KPI: `signup completion rate`.
- Срезы: устройство, источник трафика, шаг воронки.
- На дашборде выявлен провал на шаге подтверждения телефона.
- Решение: упростить форму и добавить альтернативный канал OTP.

## Инструменты

| Инструмент | Сильные стороны | Ограничения |
| --- | --- | --- |
| Tableau | сильный storytelling и быстрые прототипы | стоимость и governance в enterprise |
| Power BI | экосистема Microsoft, self-service | сложные DAX-выражения требуют экспертизы |
| Looker | централизованный metric layer | требует зрелого DWH-контура |
| Metabase | быстрый старт и простота | ограниченные advanced-возможности |
