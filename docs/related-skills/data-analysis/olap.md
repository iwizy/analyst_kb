# OLAP

OLAP (Online Analytical Processing) - подход к аналитической обработке данных,
ориентированный на агрегирование, срезы и многомерный анализ.

## Ключевые признаки

- сложные запросы по большим объемам исторических данных;
- агрегаты и измерения (время, продукт, регион и т.д.);
- фокус на чтении, а не на транзакционных изменениях;
- поддержка drill-down и сравнения периодов.

## Где полезно аналитику

- анализ динамики метрик и трендов;
- сегментация пользователей и продуктовых сценариев;
- оценка влияния изменений на KPI.

## Практический пример

Для анализа конверсии по воронке аналитик использует OLAP-витрину:

- измерения: канал, дата, сегмент клиента;
- метрики: регистрации, заявки, конверсия;
- операции: сравнение периодов и drill-down по сегментам.

## Лучшие практики

- формировать витрины под конкретные управленческие вопросы;
- фиксировать бизнес-определения метрик в словаре;
- разделять слой фактов и слой представления для отчетов.

## Типичные ошибки

- разные определения одной метрики в разных отчетах;
- отсутствие контроля качества данных в источниках;
- перегрузка витрины избыточными полями без потребности.

## Чек-лист качества

- Метрики имеют единые формулы и владельцев.
- Витрина покрывает ключевые разрезы анализа.
- Есть проверка качества и актуальности загрузок.
