# Анализ данных

Анализ данных помогает системному аналитику проверять гипотезы, уточнять
требования на основе фактов и измерять эффект изменений.

## Где это применяется

- проверка проблем и гипотез до запуска разработки;
- формирование KPI и критериев успеха фичи;
- поиск причин отклонений в процессах и метриках;
- подготовка аргументированных решений для стейкхолдеров.

## Базовый рабочий цикл

1. Сформулировать вопрос и метрику.
1. Определить источник и качество данных.
1. Выполнить анализ и проверить допущения.
1. Подготовить выводы и рекомендации для решения.
