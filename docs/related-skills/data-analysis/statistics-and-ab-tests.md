# Статистика и A/B-тесты

Базовая статистика нужна аналитику, чтобы проверять гипотезы корректно, а не по случайным колебаниям.

## Минимальный набор понятий

- среднее, медиана, дисперсия;
- доверительный интервал;
- статистическая значимость (p-value);
- эффект (uplift) и practical significance.

## A/B-тест: базовый процесс

1. Сформулировать гипотезу и primary metric.
1. Рассчитать размер выборки и длительность теста.
1. Рандомизировать пользователей и исключить смещения.
1. Оценить результат по значимости и бизнес-эффекту.
1. Принять решение: rollout / iterate / stop.

## Типичные ошибки

- остановка теста до достижения целевой выборки;
- множественные проверки без корректировки;
- выводы только по p-value без оценки эффекта.
