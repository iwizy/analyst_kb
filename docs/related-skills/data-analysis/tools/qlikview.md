# QlikView

> Глоссарий терминов и сокращений: [Открыть](../../../glossary.md)


QlikView полезен для исследовательской аналитики благодаря ассоциативной модели данных и быстрым ad-hoc срезам.

## Уровни сложности

### Базовый уровень

- подключать источники и собирать листы;
- использовать селекции и фильтры;
- понимать ассоциативные связи.

### Средний уровень

- проектировать модель данных без двусмысленных полей;
- создавать вычисляемые меры;
- документировать логику расчетов.

### Продвинутый уровень

- оптимизировать производительность и reload;
- внедрять governance и роли доступа;
- стандартизировать библиотеку KPI и шаблонов.

## Когда выбирать QlikView

- важен глубокий ad-hoc анализ;
- в компании уже есть Qlik-экосистема;
- нужно исследовать множество пересекающихся измерений.

## Плюсы и ограничения

| Плюсы | Ограничения |
| --- | --- |
| мощная ассоциативная аналитика | выше порог обучения |
| быстрый deep-dive по данным | требуется дисциплина модели |
| гибкие селекции | governance нужно настраивать отдельно |
