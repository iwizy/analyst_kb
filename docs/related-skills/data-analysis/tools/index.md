# Инструменты анализа данных

Инструменты BI и аналитики помогают превращать данные в управленческие решения:
строить отчеты, дашборды и аналитические срезы.

## Задачи, которые решают инструменты

- визуализация ключевых метрик и трендов;
- self-service аналитика для бизнеса;
- контроль выполнения KPI;
- быстрый анализ отклонений по сегментам.

## Ключевые решения в разделе

- [Power BI](power-bi.md)
- [QlikView](qlikview.md)
- [Tableau](tableau.md)

## Критерии выбора инструмента

- экосистема и интеграции с источниками;
- удобство моделирования данных;
- возможности визуализации и интерактивности;
- governance, безопасность и лицензирование.
