# Tableau

> Глоссарий терминов и сокращений: [Открыть](../../../glossary.md)


Tableau подходит для интерактивной визуализации, exploratory analysis и быстрого storytelling для бизнеса.

## Уровни сложности

### Базовый уровень

- подключить источник данных;
- собрать базовый дашборд;
- настроить фильтры и tooltip.

### Средний уровень

- использовать calculated fields и параметры;
- собирать dashboard actions;
- оптимизировать производительность визуализаций.

### Продвинутый уровень

- внедрять governance (roles, certified data sources);
- стандартизировать визуальный стиль и KPI;
- строить reusable templates для команд.

## Базовый workflow

1. Подключить источник (DWH/CSV/API).
2. Проверить типы полей и гранулярность.
3. Создать листы под ключевые вопросы.
4. Собрать дашборд и фильтры.
5. Опубликовать и назначить владельца.

## Плюсы и ограничения

| Плюсы | Ограничения |
| --- | --- |
| сильная визуализация | стоимость лицензий |
| быстрый prototyping | сложный governance без процесса |
| удобный storytelling | нужна дисциплина метрик |
