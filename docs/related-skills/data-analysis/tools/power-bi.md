# Power BI

Power BI - BI-платформа Microsoft для моделирования данных, построения отчетов
и дашбордов.

## Когда особенно полезен

- организация использует стек Microsoft;
- нужны управляемые корпоративные дашборды;
- важна интеграция с Excel, SQL Server, Azure.

## Ключевые компоненты

- Power Query: загрузка и преобразование данных;
- Data Model: связи таблиц и меры;
- DAX: вычисления и бизнес-метрики;
- Service: публикация, доступ и обновление.

## Практический сценарий аналитика

1. Собрать данные из CRM и биллинга.
1. Построить модель и меры по ключевым KPI.
1. Сформировать дашборд для бизнес-пользователей.
1. Настроить расписание обновления и контроль качества.

## Лучшие практики

- star-schema для модели данных;
- вынос бизнес-логики в меры, а не в визуализации;
- единый словарь KPI и формул;
- контроль производительности сложных DAX-выражений.

## Типичные ошибки

- перегрузка дашборда визуализациями;
- дублирующиеся меры с разной логикой;
- отсутствие управления доступом к чувствительным данным.

## Чек-лист качества

- Метрики согласованы и интерпретируются однозначно.
- Дашборд отвечает на конкретные управленческие вопросы.
- Обновления стабильны и контролируются.
