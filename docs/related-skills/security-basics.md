# Основы безопасности

Базовая security-грамотность нужна аналитику, чтобы не закладывать уязвимости на этапе требований.

## Минимальный набор тем

- аутентификация и авторизация;
- защита данных в transit/at rest;
- OWASP Top 10;
- секреты и ключи доступа;
- secure-by-design практики.

## Практический пример

Сценарий: API личного кабинета.

- риск: токены доступа без срока жизни;
- последствие: компрометация сессии;
- требование: короткий access token + refresh token rotation + mTLS для межсервисных вызовов.

## Практические вопросы к требованиям

- какие данные чувствительные;
- кто и на каком основании получает доступ;
- как логируются и аудируются критичные действия;
- какие есть сценарии злоупотреблений.
