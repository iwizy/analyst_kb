# Пользовательский интерфейс (UI/UX)

UI/UX в архитектурном контексте - это проектирование пользовательского опыта,
который одновременно удобен для пользователя и реализуем в ограничениях системы.

## Роль аналитика в UI/UX

- переводить бизнес-цели в сценарии пользовательского поведения;
- выявлять точки потерь и ошибки в пути пользователя;
- синхронизировать UX-решения с техническими ограничениями и SLA.
