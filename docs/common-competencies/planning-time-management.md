# Планирование и управление временем

Эффективное планирование снижает хаос в приоритетах и повышает предсказуемость поставки.

## Практики

- time blocking для deep work;
- weekly planning с фокусом на outcomes;
- daily reprioritization по входящим изменениям;
- ограничение WIP для аналитических задач.

## Шаблон недельного плана

| Цель недели | Ключевые задачи | Риски/блокеры | Митигирующие действия |
| --- | --- | --- | --- |
| Подготовить релизную спецификацию | API review, NFR, criteria | задержка от security | ранняя синхронизация |

## Типичные ошибки

- перегруз задачами без буфера;
- отсутствие приоритизации по ценности;
- игнорирование времени на коммуникации и review.
