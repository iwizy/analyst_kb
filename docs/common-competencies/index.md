# Общие компетенции

Раздел объединяет универсальные практики системного аналитика: оценка задач, риск-менеджмент, финансовое обоснование, работа со стейкхолдерами и принятие решений на основе данных.

## Что покрыто

- оценка трудоемкости и прогнозирование сроков;
- выявление и управление рисками;
- ROI/NPV для обоснования инициатив;
- планирование и управление временем;
- управление стейкхолдерами;
- data-driven подход к решениям.
