# Affinity Diagram

Affinity Diagram (диаграмма сродства) помогает сгруппировать большое число идей/наблюдений в смысловые кластеры.

## Когда использовать

- после интервью/воркшопа с большим объемом заметок;
- при формировании тем для backlog;
- для подготовки к приоритизации.

## Процесс

1. Собрать атомарные наблюдения на карточках.
2. Без обсуждения сгруппировать по сходству.
3. Назвать кластеры.
4. Согласовать интерпретации.
5. Превратить кластеры в задачи/гипотезы.

## Чек-лист

- кластеры имеют понятные названия;
- нет "мусорной" группы "разное" без анализа;
- результаты конвертированы в backlog.

## Контрольные вопросы

1. Какие кластеры дают наибольшую ценность для roadmap?
2. Какие наблюдения не вошли в кластеры и почему?
3. Какие решения можно принять уже после первой сессии affinity?
