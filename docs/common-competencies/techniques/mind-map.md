# Mind Map

Mind Map — техника визуальной декомпозиции темы от центральной цели к веткам и подветкам.

## Уровни сложности

### Базовый уровень

- строить карту для сбора контекста;
- выделять ключевые ветки предметной области;
- фиксировать пробелы и вопросы.

### Средний уровень

- использовать карту для подготовки интервью/воркшопов;
- переводить ветки в требования и backlog;
- поддерживать версионность карты.

### Продвинутый уровень

- связывать карту с архитектурой и рисками;
- использовать mind map для портфельной декомпозиции;
- интегрировать карту в систему знаний команды.

## Пример структуры карты (финтех)

- Онлайн-кредитование
- Клиентский путь
- Скоринг и антифрод
- Решение и договор
- Сопровождение и взыскание

## Пошаговый сценарий

1. Определить центральную тему.
2. Добавить ветки первого уровня (MECE по возможности).
3. Уточнить подветки до actionable уровня.
4. Отметить open questions.
5. Конвертировать ветки в задачи/требования.
