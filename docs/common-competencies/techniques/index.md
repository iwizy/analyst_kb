# Техники и инструменты анализа

Техники анализа помогают структурировать проблему, сформировать гипотезы, согласовать решения и превратить выводы в управляемые действия.

## Уровни сложности

### Базовый уровень

- применять базовые техники для диагностики и декомпозиции;
- фиксировать выводы в шаблонном виде;
- связывать результаты с backlog.

### Средний уровень

- комбинировать техники под контекст задачи;
- проводить фасилитационные сессии;
- использовать визуальные инструменты и шаблоны.

### Продвинутый уровень

- выстраивать библиотеку техник на уровне команды/программы;
- измерять эффективность применения техник;
- интегрировать техники в регулярный SDLC-ритм.
