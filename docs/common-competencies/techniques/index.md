# Техники

Техники анализа помогают быстро структурировать проблему, сформировать гипотезы
и принять обоснованное решение в условиях ограниченного времени.

## Когда применять

- на discovery-этапе для формулирования гипотез;
- на refinement для уточнения границ и причин проблем;
- на ретроспективах и postmortem для анализа первопричин.
