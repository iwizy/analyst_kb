# ГОСТ 19

ГОСТ 19 (ЕСПД) - комплекс стандартов программной документации. Он задает типы
документов и правила их оформления для программных средств.

## Когда применяют

- проекты требуют формализованного набора программных документов;
- есть контрактные/отраслевые требования к формату документации;
- важна стандартизированная структура документации в организации.

## Что обычно используется аналитиком

- описание программы и ее назначения;
- правила оформления пояснительной документации;
- требования к структуре эксплуатационных документов.

## Практическое применение

- согласовать обязательный набор документов по контракту;
- адаптировать шаблоны под требования ГОСТ 19;
- связать документы с артефактами жизненного цикла разработки.

## Типичные ошибки

- формальное соответствие без реальной полезности документов;
- дублирование информации в нескольких ГОСТ-документах;
- отсутствие актуализации документов после изменений.

## Чек-лист качества

- Набор документов соответствует обязательным требованиям.
- Структура и реквизиты выдержаны по стандарту.
- Документы актуальны и связаны с фактической реализацией.
