# Выявление рисков

Выявление рисков - системный поиск событий, способных повлиять на сроки, стоимость, качество и ценность результата.

## Матрица рисков

Матрица `Вероятность x Влияние` помогает ранжировать риски.

| Вероятность \ Влияние | Низкое | Среднее | Высокое |
| --- | --- | --- | --- |
| Низкая | Monitor | Monitor | Control |
| Средняя | Control | Mitigate | Priority |
| Высокая | Priority | Escalate | Escalate |

## FMEA (Failure Mode and Effects Analysis)

Для каждого риска оцениваются:

- Severity (S),
- Occurrence (O),
- Detection (D),
- `RPN = S * O * D`.

## SWOT в риск-контуре

SWOT помогает обнаружить внешние угрозы и внутренние слабости, которые трансформируются в риски проекта.

## Типичные ошибки

- фиксировать риск без владельца;
- путать риск и уже случившуюся проблему;
- не обновлять риск-реестр после изменений.
