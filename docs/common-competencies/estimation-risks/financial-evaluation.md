# Финансовая оценка инициатив (ROI/NPV)

Финансовая оценка помогает приоритизировать инициативы по экономической ценности.

## ROI

`ROI = (Benefit - Cost) / Cost * 100%`

Подходит для быстрой сравнительной оценки.

## NPV

`NPV = Σ(CashFlow_t / (1 + r)^t) - InitialInvestment`

Учитывает стоимость денег во времени.

## Пример таблицы

| Показатель | Инициатива A | Инициатива B |
| --- | --- | --- |
| Инвестиции | 8 млн | 5 млн |
| Годовая выгода | 3 млн | 2 млн |
| Горизонт | 4 года | 4 года |
| ROI | 50% | 60% |
| NPV (r=12%) | 1.1 млн | 0.9 млн |

Интерпретация: B лучше по ROI, A лучше по NPV.

## Практические рекомендации

- считать сценарии base/optimistic/pessimistic;
- документировать допущения;
- учитывать риск-премию в ставке дисконтирования.
