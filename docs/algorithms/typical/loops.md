# Циклы и базовая итеративная обработка

Циклы - основа большинства алгоритмов обработки данных. В прикладных системах от корректности цикла зависят производительность, полнота обработки и устойчивость к ошибкам.

## Когда использовать

- последовательный проход по записям;
- фильтрация и агрегация;
- валидация пакетов данных;
- обработка очередей и батчей.

## Типовые конструкции

- `for`: когда известен набор или диапазон.
- `while`: когда повторяем до выполнения условия.
- `break/continue`: управляем остановкой и пропуском.

## Псевдокод: суммирование валидных платежей

```text
sumValidPayments(payments):
  total = 0
  for each p in payments:
    if p.status != "SUCCESS":
      continue
    total = total + p.amount
  return total
```

Сложность: `O(n)` по времени и `O(1)` по памяти.

## Бизнес-пример

Финансовый batch: нужно обработать 3 млн транзакций и выделить только успешные для отчета и сверки.

## Типичные ошибки

- бесконечный цикл из-за неверного условия;
- модификация коллекции во время прохода;
- вложенные циклы `O(n^2)` вместо использования индекса/хеша.

## Чек-лист

- есть гарантированное условие завершения;
- покрыты пустой вход и граничные значения;
- известна асимптотика и влияние на SLA.
