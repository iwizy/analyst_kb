# Карта типовых
 алгоритмов

Подраздел объединяет алгоритмы, которые чаще всего встречаются в прикладных системах и собеседованиях.

## Уровни сложности

### Базовый

- понимать базовые паттерны: сортировка, поиск, обход;
- решать задачи на массивы, строки, очереди;
- оценивать сложность базовых решений.

### Средний

- комбинировать структуры и алгоритмы;
- выбирать подход под профиль данных;
- объяснять trade-off решения.

### Продвинутый

- переходить к DP, жадным и графовым стратегиям;
- проектировать масштабируемые алгоритмические pipeline.

## Задачи для самопроверки

1. Для каждого типа задачи из таблицы подберите альтернативный алгоритм.
2. Оцените, в каких случаях альтернатива будет хуже по памяти.
