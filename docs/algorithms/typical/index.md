## Матрица выбора

| Задача | Что использовать |
| --- | --- |
| Проверить наличие элемента в небольшом списке | Линейный поиск |
| Быстрый поиск в отсортированном массиве | Бинарный поиск |
| Упорядочить большой набор с предсказуемым временем | Merge sort / Heap sort |
| Быстрая средняя сортировка без строгой стабильности | Quick sort |
| Обход связей и зависимостей | BFS / DFS |
| Кратчайший маршрут в взвешенном графе | Dijkstra |
| Быстрые операции ключ -> значение | Hash table |
| Иерархические данные с быстрым поиском/вставкой на диске | B-tree |

## Практический принцип

Сначала определяйте профиль нагрузки и ограничения (время, память, консистентность), затем выбирайте алгоритм.
