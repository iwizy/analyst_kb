# Теория чисел (продвинутый уровень)

> Глоссарий терминов и сокращений: [Открыть](../../glossary.md)


Алгоритмы теории чисел важны для криптографии, контрольных сумм, генерации ключей и оптимизации вычислений.

## Уровни сложности

### Базовый

- понимать GCD и простые числа;
- использовать быстрое возведение в степень;
- применять решето Эратосфена.

### Средний

- работать с modular arithmetic;
- применять расширенный алгоритм Евклида;

### Продвинутый

- применять алгоритмы в криптографии и протоколах;
- оценивать сложность и безопасность реализаций.

## Ключевые алгоритмы

### Евклидов алгоритм (GCD)

```text
gcd(a, b):
  while b != 0:
    (a, b) = (b, a mod b)
  return a
```

- Сложность: `O(log min(a,b))`.

### Решето Эратосфена

```text
sieve(n):
  is_prime[0..n] = true
  is_prime[0], is_prime[1] = false
  for p in 2..sqrt(n):
    if is_prime[p]:
      for x in p*p..n step p:
        is_prime[x] = false
```

- Сложность: `O(n log log n)`.

### Быстрое возведение в степень

```text
pow_mod(a, e, m):
  res = 1
  while e > 0:
    if e odd: res = (res * a) mod m
    a = (a * a) mod m
    e = e // 2
  return res
```

- Сложность: `O(log e)`.

### Китайская теорема остатка (CRT)

- Позволяет восстановить число по системе сравнений при взаимно простых модулях.
- Используется в криптографии и оптимизации modular вычислений.
