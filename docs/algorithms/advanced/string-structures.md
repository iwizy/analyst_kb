# Строковые структуры и алгоритмы (продвинутый уровень)

> Глоссарий терминов и сокращений: [Открыть](../../glossary.md)


Строковые структуры нужны для поиска по текстам, логам, справочникам и биоинформатике.

## Уровни сложности

### Базовый

- понимать Trie и префиксный поиск;
- знать KMP и hash-based matching.

### Средний

- использовать Aho-Corasick для множественных шаблонов;
- понимать суффиксный массив как индекс строки.

### Продвинутый

- применять суффиксные деревья/автоматы для сложного matching;
- оптимизировать память индексов.

## Ключевые структуры

| Структура | Для чего |
| --- | --- |
| Trie | автодополнение, словари, префиксы |
| Aho-Corasick automaton | поиск множества шаблонов за один проход |
| Suffix array | быстрые substring-запросы |
| Suffix tree | мощный, но более тяжелый индекс |

## Пример Trie-вставки

```text
insert(word):
  node = root
  for ch in word:
    if node.next[ch] missing: create
    node = node.next[ch]
  node.is_terminal = true
```
