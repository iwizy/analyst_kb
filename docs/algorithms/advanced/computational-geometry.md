# Геометрические алгоритмы (продвинутый уровень)

Геометрические алгоритмы применяются в GIS, робототехнике, CAD, навигации и анализе пространственных данных.

## Уровни сложности

### Базовый

- ориентироваться в векторных операциях;
- использовать orientation/cross product;
- решать задачи пересечения отрезков.

### Средний

- строить выпуклую оболочку (Graham scan);
- применять sweep line для событий на плоскости.

### Продвинутый

- использовать триангуляцию и продвинутые структуры;
- анализировать числовую устойчивость и precision issues.

## Ключевые алгоритмы

| Алгоритм | Сложность | Применение |
| --- | --- | --- |
| Orientation / cross product | O(1) | поворот, пересечение |
| Graham scan | O(n log n) | выпуклая оболочка |
| Sweep line | O((n + k) log n) | пересечение отрезков |
| Триангуляция | зависит от метода | mesh, графика |

## Пример: orientation

```text
orientation(a, b, c):
  val = (b.x-a.x)*(c.y-a.y) - (b.y-a.y)*(c.x-a.x)
  if val > 0: left turn
  if val < 0: right turn
  else: collinear
```
