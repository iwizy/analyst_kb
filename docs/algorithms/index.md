# Алгоритмы

Раздел покрывает алгоритмические основы, которые помогают системному аналитику
корректно обсуждать производительность, ограничения и технические компромиссы
при проектировании решений.

## Что дает системному аналитику

- обоснованный диалог с разработкой о трудоемкости и SLA;
- понимание, где узкие места связаны с алгоритмом, а где с инфраструктурой;
- более качественные нефункциональные требования по времени и памяти;
- возможность заранее оценивать риски роста нагрузки.

1. Сверьтесь с [областями использования](usage-areas.md), чтобы выбрать решение
   под конкретный бизнес-сценарий.
