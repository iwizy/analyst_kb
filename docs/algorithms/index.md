# Алгоритмы

Раздел оформлен как короткий курс по алгоритмам и структурам данных для системного аналитика. Цель курса - уметь выбирать подход к обработке данных и обосновывать требования по производительности.

## Зачем аналитику алгоритмы

- оценивать реалистичность SLA и NFR до реализации;
- понимать, где проблема в алгоритме, а где в инфраструктуре;
- обсуждать компромиссы с разработкой на общем языке;
- снижать риск неверных архитектурных решений при росте нагрузки.

## Структура курса

## Что вы получите после прохождения

- понимание разницы между `O(n)`, `O(log n)`, `O(n log n)`, `O(n^2)`;
- практический выбор сортировки, поиска, обходов графов и структур хранения;
- базовый алгоритмический словарь для обсуждения архитектуры и интеграций;
- набор задач для закрепления материала.
