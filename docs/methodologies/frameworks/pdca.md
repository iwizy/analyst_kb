# PDCA

> Глоссарий терминов и сокращений: [Открыть](../../glossary.md)


PDCA (Plan-Do-Check-Act) - цикл непрерывного улучшения процессов. Подходит для точечных и системных изменений в delivery-контуре.

## Уровни сложности

### Базовый уровень

- понимать 4 шага PDCA;
- применять цикл к одной проблеме процесса;
- фиксировать метрику до/после.

### Средний уровень

- запускать серию PDCA-улучшений по квартальному плану;
- внедрять стандарты по результатам успешных циклов;
- интегрировать PDCA в ретроспективы и governance.

### Продвинутый уровень

- использовать PDCA на уровне нескольких команд;
- строить систему process KPI и improvement backlog;
- сочетать PDCA с Lean/Kanban-практиками.

## Цикл PDCA

```kroki-plantuml
skinparam monochrome true
start
:Plan - цель и гипотеза;
:Do - пилотное внедрение;
:Check - измерение результата;
if (Цель достигнута?) then (Да)
  :Act - стандартизация;
else (Нет)
  :Act - корректировка и новый цикл;
endif
stop
```

## Артефакты

| Артефакт | Назначение |
| --- | --- |
| Improvement Hypothesis | гипотеза улучшения |
| Baseline Metrics | исходная метрика |
| Experiment Log | ход пилота |
| Standard Update | закрепление результата |

## Пример применения

Цель: снизить lead time согласования требований.  
Plan: внедрить шаблон требований и SLA review.  
Do: пилот на одной команде 2 спринта.  
Check: сравнить до/после.  
Act: масштабировать или скорректировать подход.

## Плюсы и ограничения

Плюсы:

- системный подход к улучшениям;
- решения на основе данных;
- хорошо сочетается с любыми фреймворками.

Ограничения:

- требует дисциплины измерений;
- без owner и сроков быстро деградирует в формальность.

## Связь с требованиями и рисками

- PDCA помогает улучшать процессы elicitation/validation;
- риск-триггеры можно включать в этап Plan;
- результаты цикла должны отражаться в рабочих стандартах.

## Типичные ошибки

- запуск улучшений без baseline;
- слишком широкий масштаб пилота;
- отсутствие стандартизации после успешного результата.## Источники

- Deming cycle references.
- Lean continuous improvement practices.
