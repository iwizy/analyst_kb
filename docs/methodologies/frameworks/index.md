## Быстрый выбор

| Контекст | Базовый выбор |
| --- | --- |
| 1 команда, продуктовый цикл | Scrum / Kanban |
| Нужен поток непрерывных задач | Kanban |
| Нужна инженерная дисциплина | XP |
| Крупная программа 5+ команд | SAFe / LeSS |
| Сложный домен | DDD + Scrum/Kanban |
| Функциональная декомпозиция по фичам | FDD |
