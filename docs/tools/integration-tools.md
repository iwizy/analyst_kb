# Инструменты интеграций

## Популярные решения

| Инструмент | Когда использовать | Плюсы | Минусы |
| --- | --- | --- | --- |
| Talend | ETL/интеграционные пайплайны | визуальная сборка потоков, много коннекторов | enterprise-сложность и стоимость |
| Apache NiFi | потоковая маршрутизация данных | гибкая orchestration data flow | требует операционной экспертизы |
| Kafka UI / Conduktor | работа с Kafka-топиками | прозрачность событийных потоков | не заменяет системный мониторинг |
| SoapUI | контрактные интеграционные тесты | силен для SOAP/XML | частично дублируется с Postman в REST |

## Как начать

1. Опишите интеграционные паттерны и SLA обмена.
1. Настройте контракты и schema governance.
1. Введите мониторинг задержек, ошибок, повторов.
1. Проверьте recovery-сценарии (retry, DLQ, replay).
