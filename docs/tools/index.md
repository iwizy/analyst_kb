# Инструменты

Раздел объединяет практические инструменты системного аналитика для работы с
требованиями, моделями, API, документацией и исследованием поведения систем.

## Что дает этот раздел

- ускоряет выбор подходящего инструмента под конкретную задачу;
- помогает выстроить единый рабочий процесс с разработкой, QA и бизнесом;
- снижает количество ошибок в артефактах за счет проверенных практик.

## Как использовать раздел

1. Определите цель: моделирование, API-анализ, документация, прототипирование.
1. Выберите инструмент с учетом контекста команды и процесса поставки.
1. Примените базовые практики и чек-листы из статьи инструмента.
1. Свяжите артефакт инструмента с требованиями и решением по задаче.
