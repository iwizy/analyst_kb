# Шаблон RTM

RTM помогает контролировать трассировку от бизнес-цели до тестов и реализации.

## Таблица трассировки

| BR | UR/SR | Feature/Task | Test Case | Статус | Комментарий |
|---|---|---|---|---|---|
| BR-001 | UR-003, SR-014 | FEAT-29, DEV-182 | TC-104, TC-105 | In Progress | |

## Минимальные правила

- каждая строка должна иметь владельца;
- не оставлять orphan-требования без задач и тестов;
- обновлять связи при каждом Change Request.

## Метрики контроля

| Метрика | Формула | Цель |
|---|---|---|
| Coverage | Требования с тестами / Все требования | >= 95% |
| Orphan count | Требования без связей | 0 в релизе |
| Actuality | Актуализированные связи / Все связи | >= 95% |
