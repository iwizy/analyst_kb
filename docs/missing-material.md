## Правило обновления глоссария

При добавлении новых терминов/сокращений в материалы:

1. Добавить определение в `docs/glossary.md`.
2. Проверить алфавитную сортировку терминов и сокращений.
3. Проставить ссылки на глоссарий в тексте только там, где это не ломает существующие markdown-ссылки.
4. Проверить сборку `mkdocs build --strict`.
