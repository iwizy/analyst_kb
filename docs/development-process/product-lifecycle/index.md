# Жизненный цикл программного продукта

Жизненный цикл ПО (SDLC) задает последовательность работ и артефактов, которые превращают идею в устойчиво работающий продукт.

## Ключевые блоки

- [Основы DevOps](devops-basics.md)
- [Основы CI/CD](cicd-basics.md)
- [Git](git.md)

## Сквозной цикл

```kroki-plantuml
@startuml
start
:Инициация;
:Анализ;
:Дизайн;
:Разработка;
:Тестирование;
:Развертывание;
:Эксплуатация;
:Улучшения;
stop
@enduml
```

## Что важно аналитику

- на каждом этапе есть обязательные входы/выходы;
- качество артефактов влияет на стоимость следующих этапов;
- неполные требования и риски на старте почти всегда дороже в конце.
