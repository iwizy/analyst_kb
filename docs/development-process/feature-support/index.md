# Сопровождение функциональности

Сопровождение функциональности включает эксплуатацию, мониторинг, реакцию на
инциденты и постоянное улучшение стабильности продукта.

## Задачи сопровождения

- поддерживать целевые SLA/SLO;
- быстро обнаруживать и устранять отклонения;
- предотвращать повторение инцидентов через системные улучшения.

## Практическое дополнение

- Проведите мини-аудит текущего процесса по схеме: `требование -> реализация -> тест -> эксплуатация`.
- Зафиксируйте одно узкое место и добавьте улучшение в ближайший спринт/итерацию.
- Проверьте эффект через 2 недели по измеримой метрике (дефекты, lead time, MTTR или SLA).
